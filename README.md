# ejercicio-js-con-ui

## Setup del entorno

```bash
# instalar nodejs y npm
sudo yum install nodejs
curl http://npmjs.org/install.sh | sh

# instalar grunt-cli y bower globalmente
npm install -g grunt-cli
npm install -g bower
```
